# Changelog

# 1.1.0 (2020-07-30)

# Code Refactoring

  • Global $routerTab is changed to $tabs (6edc214)

# Features

  • contextmenu: Custom contextmenu support (d475ebe)
  • beforePageLeave Support browser beforeunload event (5d57e60)

# BREAKING CHANGES

  • Global access to the RouterTab component this.$routerTab has been deprecated and changed to this.$tabs
  • beforePageLeave deprecate the parameters resolve, reject, return a promise to allow or prevent the tab page from leaving

# 1.0.0 (2020-06-28)

# Code Refactoring

  • Style refactoring, remove left and right tab scroll buttons (a17543a)

  • iframe: Remove openIframeTab, closeIframeTab, refreshIframeTab methods (87d0d77)

# Features

  • RouterAlive: RouterAlive component is refactored using keep-alive + router-view (ac3a04d)
  • scroll: Add tab scroll component (620d17e)-
  • slot: New Slots: start and end (34d8542)
  • tab: Mew route meta option: closable and tabClass (b6c92fb)

# Performance Improvements

  • Cached nested pages are not updated, use $forceUpdate (a85a51e)
  • RouterAlive: Match page route depth by $vnode.data.routerViewDepth (c0dce87)

# BREAKING CHANGES

  • Component node class change
  • tab: Tab slot scope variable changes
  • iframe: Remove openIframeTab, closeIframeTab, refreshIframeTab methods
  • RouterAlive:
    1. Simplify the close method arguments: close(path, to)
    2. Deprecated options：alive-id、router-view
    3. Change meta option aliveId to key
    4. Deprecated routing page mark _isRouterPage
